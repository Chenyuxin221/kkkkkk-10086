# 0.1.3
* 添加命令 `#kkk删除缓存`
  * 指向视频缓存(目录：/resources/kkkdownload/video/)，当缓存过多可使用该命令一键删除所有缓存视频

# 0.1.2
* 抖音添加 `直播` 类型推送

# 0.1.1
* 添加 `#抖音强制推送` `#B站强制推送`
  * 只在触发群进行推送，其他群不推送。如果配置的推送的用户过多，可能导致刷屏
